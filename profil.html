Hier teste ich Funktionen in HTML, CSS und JS aus

Rausfinden, was HTML, CSS & JS überhaupt ist.

HTML

HTML kann man sich wie das Skelett eines Menschen vorstellen. Es bildet die grundlegende Struktur einer Webseite und definiert, welche Elemente vorhanden sind, zum Beispiel Überschriften, Texte und Bilder.

CSS

CSS ist für das Aussehen der Webseite zuständig. Es bestimmt Farben, Abstände und Layout. Man könnte sagen: CSS ist wie die äussere Erscheinung oder das Design eines Körpers.

JavaScript

Was wäre also eine Fleischhülle ohne Reaktionen und Empfinden? JavaScript bringt die Webseite zum Leben. Es sorgt für Interaktivität und Reaktionen, beispielsweise bei Klicks oder Animationen. Ohne JS wäre eine Webseite statisch.
