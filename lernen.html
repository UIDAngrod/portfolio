HTML SHEET

HTML

HTML kann man sich wie das Skelett eines Menschen vorstellen. Es bildet die grundlegende Struktur einer Webseite und definiert, welche Elemente vorhanden sind, zum Beispiel Überschriften, Texte und Bilder.

GRUNDSTRUKTUR

  • <html> – gesamte Seite
  • <head> – Metadaten
  • <body> – sichtbarer Inhalt
  • <title> – Titel im Tab

TEXT

  • <h1> - <h6> – Überschriften
  • <p> – Absatz
  • <br> – Zeilenumbruch
  • <strong> – fett
  • <em> – kursiv

LINKS

  • <a href=""> – Link erstellen
  • target="_blank" – neuer Tab
  • mailto: – E-Mail Link

BILDER

  • <img src=""> – Bild anzeigen
  • alt="" – Beschreibung
  • width / height – Grösse

LISTEN

  • <ul> – ungeordnete Liste
  • <ol> – nummerierte Liste
  • <li> – Listenelement

LAYOUT

  • <div> – Container
  • <span> – Inline Container
  • <section> – Abschnitt
  • <header> – Kopfbereich
  • <footer> – Fussbereich

FORMULARE

  • <form> – Formular
  • <input> – Eingabe
  • <button> – Button
  • <label> – Beschriftung

ATTRIBUTE

  • class – CSS Klasse
  • id – eindeutige ID
  • style – Inline CSS
  • href / src – Pfade

CSS SHEET

CSS

CSS ist für das Aussehen der Webseite zuständig. Es bestimmt Farben, Abstände und Layout. Man könnte sagen: CSS ist wie die äussere Erscheinung oder das Design eines Körpers.

FONT

  • font-family – Schriftart ändern
  • font-style – normal, kursiv oder schräg
  • font-weight – Schriftstärke / fett
  • font-size – Schriftgrösse
  • font – Kurzform für mehrere Schriftwerte

FARBEN & HINTERGRUND

  • color – Textfarbe
  • background-color – Hintergrundfarbe
  • background-image – Hintergrundbild
  • background-repeat – Bild wiederholen
  • background – Kurzform

TEXT

  • word-spacing – Abstand zwischen Wörtern
  • letter-spacing – Abstand zwischen Buchstaben
  • text-decoration – Unterstrich / durchgestrichen
  • text-align – links, mittig, rechts
  • line-height – Zeilenabstand

LISTEN

  • list-style-type – Art der Bulletpoints
  • list-style-image – Bild als Bulletpoint
  • list-style-position – Position der Listenpunkte

BOX MODELL

  • margin – Aussenabstand
  • padding – Innenabstand
  • border – Rahmen
  • width – Breite
  • height – Höhe

LAYOUT

  • float – Element links/rechts schweben lassen
  • clear – Float aufheben
  • margin: 0 auto; – Box horizontal zentrieren
  • text-align: center; – Text zentrieren

JAVASCRIPT SHEET

JavaScript

Was wäre also eine Fleischhülle ohne Reaktionen und Empfinden? JavaScript bringt die Webseite zum Leben. Es sorgt für Interaktivität und Reaktionen, beispielsweise bei Klicks oder Animationen. Ohne JS wäre eine Webseite statisch.

VARIABLEN

  • let – veränderbare Variable
  • const – konstante Variable
  • var – alte Schreibweise

DATENTYPEN

  • string – Text
  • number – Zahl
  • boolean – true / false
  • array – Liste
  • object – Objekt

FUNKTIONEN

  • function – Funktion erstellen
  • () – Parameter
  • return – Wert zurückgeben
  • => – Arrow Function

EVENTS

  • onclick – Klick
  • addEventListener – Event hinzufügen
  • mouseover – Maus darüber
  • keydown – Taste gedrückt

DOM

  • document.getElementById()
  • querySelector()
  • innerHTML – Inhalt ändern
  • style – CSS ändern

BEDINGUNGEN

  • if – Bedingung
  • else – sonst
  • switch – mehrere Fälle

SCHLEIFEN

  • for – Schleife
  • while – solange
  • forEach – Array durchlaufen

ARRAYS

  • push() – hinzufügen
  • pop() – entfernen
  • length – Länge
  • map() – verändern
